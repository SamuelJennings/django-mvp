Primary Sidebar Layout

This view demonstrates a single primary (left) sidebar configuration. The sidebar contains navigation links and supplementary information.

Introduction

The primary sidebar layout is one of the most common patterns in web applications. It provides:

  • Persistent navigation that remains visible as users browse content
  • Quick access to key sections or filters
  • Hierarchical organization of related features
  • Responsive behavior - sidebar becomes offcanvas on mobile devices

Key Features

Collapsible Sidebar

Optional collapse functionality allows users to maximize content area when needed.

Mobile Responsive

Automatically converts to offcanvas drawer on smaller screens for optimal mobile UX.

Customizable Width

Sidebar width can be configured via component attributes to match your design needs.

Flexible Content

Place any content in the sidebar - navigation, filters, widgets, or custom components.

Configuration

The primary sidebar can be customized with the following attributes:

  • primary_width - Width of the primary sidebar (default: 280px)
  • breakpoint - Bootstrap breakpoint for responsive behavior (default: md)
  • collapse_primary - Enable collapse functionality (default: false)
  • gap - Spacing between sidebar and content (default: 0)
