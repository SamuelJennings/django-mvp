Three-Column Layout

Primary sidebar (left), main content (center), secondary sidebar (right).

Responsive Design: Both sidebars become offcanvas panels on mobile devices. Use the menu icons in the toolbar to toggle them.

Features

This layout is ideal for documentation, research papers, or any content that benefits from both navigation and supplementary information.

  • Primary sidebar for navigation and table of contents
  • Secondary sidebar for metadata, related links, or additional context
  • Main content area that adapts to available space
  • Automatic responsive behavior on all devices
