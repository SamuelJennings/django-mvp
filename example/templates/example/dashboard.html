    AdminLTE 4 Integration

    Django MVP now provides a complete AdminLTE 4 admin dashboard template system for Django using django-cotton components.

    Key Features
    • AdminLTE Layout System - Full grid-based app-wrapper structure
    • Configuration-Driven - Control layout via MVP settings
    • Cotton Components - AdminLTE-specific reusable components
    • Bootstrap 5 - Built on Bootstrap 5 framework
